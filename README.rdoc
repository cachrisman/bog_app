== README

1. Generate the model with:
```
rails g model Creature name description

rake db:migrate
```

2. put seed data in db/seeds.rb then run

```
rake db:seed
```

3. then generate the controller actions with
```
rails g scaffold_controller Creature
```

4. add
```
root 'creatures#index'
```
to config/routes.rb

5. change each view in app/views/creature/ to add form elements or to show the creature.name and creature.description
